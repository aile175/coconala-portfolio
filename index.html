WordPressのデメリット

WordPressは優れたツールですが、すべての会社に最適とは限りません。以下の悩みがある方は、見直しをオススメします。

セキュリティーの脆弱性：WordPressは定期的なアップデートが必要です。放置すると改ざんや情報漏えいのリスクが上がるため、保守費が発生し運用コストがかかります。さらに最近では、1日当たりWordPressサイトへの攻撃が5,000万件以上発生しており、攻撃数は増加していくと見込まれる情報もあります。※お名前.comより引用

表示速度が遅い：ページ生成の仕組みや、プラグイン（簡単にテンプレートやシステムをサイトに導入できる仕組み）により、表示速度が遅くなります。表示速度に3秒以上時間がかかるとユーザーの7割は離脱（サイトを見ない）するという事がGoogleの公開情報から分かっています。また、表示速度はSEO（サイトの情報をGoogleなどのアルゴリズムが評価し、サイトを検索の上位に表示するか判定するための情報）にも影響することも分かっています。

維持・保守費だけでこんなに違います

当サービスとWordPressで「維持・保守費」だけを比べてみます。

項目 静的HP（GitHub Pages） WordPress
月額維持費 0円 6,000～18,000円
独自ドメイン費（年） 0円（github.io利用時） 約1,500円～
1年間の維持費 0円（github.io利用時） 約73,500～217,500円
5年間の維持費 0円（github.io利用時） 約367,500～1,087,500円

5年間で最大約108万円の差になるケースもあります。浮いた費用を広告や人材採用に回せます。

※ .jpなどの独自ドメイン（ホームページ固有の住所のようなもの）を使う場合は、別途ドメイン会社への年間費用が発生します。金額はドメインの種類や取得先によって異なります。

今お支払いの維持費、それに見合った効果は出ていますか？

WordPressが向いている場合

次に当てはまる場合は、WordPressのほうが適しています。

WordPressが向いているケース

  • ネット予約・ネットショップ・会員ログインなどの機能が必要な場合
  • お知らせやブログを社内スタッフが頻繁に更新したい場合
  • 専門知識なしで、管理画面から文章や画像を頻繁にご自身で直したい場合

逆に、「更新は年に数回」という場合は、当サービスの方が費用対効果が高いです。また、修正をまとめて依頼した方がコストカットできます。

なお、当サービスで制作したHP・LPの修正には、HTML/CSS（ページの見た目を作る言語）などの知識が必要なため、スポットでの保守サービスも出品しています。

更新が月4回以下なら、当サービス＋スポット保守が安くなりやすいです。

スポット保守 ― 更新は「必要なときだけ」でOK

「保守費用を抑えたい」「何かあった際のトラブルシューティングの対応をして欲しい」という方向けに、1回5,000円からのスポット保守をご用意しています。

施工実績の追加、税制の改正情報、文章修正、画像差し替えなどを必要なときだけ依頼できます。

なぜ私がWordPressを強くオススメしないのか

大きな理由は時代の変化です。AI検索の普及で、検索結果の見られ方は変化しています。最新の国内調査（推計）では、ChatGPTなどの生成AIが情報を引用するサイトのランキングなどが発表されています。そこで判明したことは、HPでブログ発信していても情報引用されずらいということです。さらに言うと、この調査結果の本質は、Web上での集客方法も大きく変化しているということです。

詳しくは以下のブログ記事で解説していますので、ぜひご一読下さい。

Q. ITが苦手でも大丈夫ですか？A. 大丈夫です。文系からIT業界に飛び込んだ経験から、難しい専門用語は使わない分かりやすい説明をいたします。

Q. 相談だけでも可能ですか？A. 可能です。目的に合わない場合は、無理な提案はしません。

Q. なぜ維持費が0円なのですか？A. GitHub Pagesというサービスを利用するからです。※独自ドメイン（.jpなど）を使用したい場合には、別途ドメイン取得・年間使用料がドメイン会社に対して発生します。アドレスにこだわりがなければ、レンタルサーバー代やドメイン使用料はかかりません。

Q. GitHub Pagesって何ですか？A. ホームページを無料で公開できるサービスです。サーバー（ホームページのデータを置く場所）の料金がかからないため、維持費を抑えやすいです。

Q. 多忙のため文章作成・画像選定から公開まで全てお任せできますか？A. 全て任せていただいて大丈夫です。制作実績にある株式会社サクラ工業様も、こちらで用意してある手間がかからずできる、デザインイメージなどを割り出すためのアンケートに答えていただき、制作させていただきました。
